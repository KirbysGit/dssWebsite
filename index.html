Welcome to SecureScape

Project Overview

SecureScape is a portable, AI-powered security system designed for remote locations without internet access. Utilizing ESP32-CAM microcontrollers equipped with infrared sensors, our system detects motion in real-time, processes images locally, and communicates alerts through a central control unit. Whether you're camping or monitoring a remote site, SecureScape offers reliable security when you need it most.

Key Features

  • AI Threat Detection Real-time object recognition using TensorFlow Lite.
  • Offline Operation Functions without internet, perfect for remote areas.
  • Energy Efficient Low-power design with long battery life.
  • User Alerts Receive notifications via our mobile app or central gadget.
  • Durable & Portable Designed to withstand outdoor conditions and easy to deploy.

How It Works

Sensor Nodes

Deployed around the perimeter, these nodes detect motion using IR sensors and capture images with ESP32-CAM modules.

Central Gadget

Collects data from nodes, processes images to identify threats, and manages alerts and system settings.

Communication

Nodes communicate wirelessly with the central gadget, creating a mesh network for reliable coverage.

Processing

AI-powered image processing runs locally on the central gadget, ensuring privacy and offline operation.
